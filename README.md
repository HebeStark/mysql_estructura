# mysql_estructura

Óptica “Cul d'Ampolla” (Modelo ER)
Este proyecto modela la base de datos de una óptica, permitiendo gestionar proveedores, marcas, gafas, clientes, empleados y ventas.

Pizzería (Modelo ER)
Este proyecto modela la base de datos de una pizzería con servicio de pedidos online, permitiendo gestionar clientes, pedidos, productos, tiendas, empleados y reparto a domicilio.

## Herramientas utilizadas

MySQL Workbench para el diseño del Diagrama Entidad–Relación (DER).
